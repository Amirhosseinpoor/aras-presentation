Step 3: Deep Dive into Information Retrieval IR Foundations
Learning how to design retrieval systems that LLM agents can trust.
  • • Studied classical IR: TF-IDF, BM25, inverted indexes
  • • Moved to dense retrieval: bi-encoders, ColBERT
  • • Explored hybrid methods: SPLADE, BM25 + SPLADE + dense + rerankers
  • • Learned IR evaluation metrics: MAP, nDCG, recall
  • • Goal: build trustworthy retrieval pipelines for LLM agents

Step 4: Occupational Health Platform (Capstone) End-to-end System
Bringing everything together: vision, IR, ML, agents, and backend into one product.
  • • Built an end-to-end AI assistant for occupational medicine
  • • Extracts data from forms and lab PDFs using OCR + vision-language models
  • • Performs disease-risk estimation with ML models
  • • Designed a hybrid IR engine: BM25, SPLADE, dense retrieval, Cross-Encoder reranking
  • • Created multi-agent workflows for generating clinician-ready reports
  • • Engineered full backend using Django, FastAPI, Celery, PostgreSQL, Docker
